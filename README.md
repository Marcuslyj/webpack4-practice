#### 学习路线：

- 基础篇（webpack的基本概念和日常开发的使用技巧）；
- 进阶篇（以工程化的方式组织webpack构建配置，和webpack打包优化）；
- 原理篇（剖析webpack打包原理和插件、loader的实现）；
- 实战篇（从实际web商城项目出发，讲解webpack实际使用）；



#### 为什么需要构建工具？

- 转换es6语法（浏览器支持不好）；
- 转换jsx；
- css前缀补全、预处理器；
- 压缩混淆；
- 图片压缩；



#### 前端构建演变之路？

ant + YUI Tool -> grunt -> gulp, fis3 -> rollup, webpack, parcel
